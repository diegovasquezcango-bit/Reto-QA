# PetStore Karate

## 1. Descripción
Automatización de pruebas API Swagger Petstore con Karate.

Flujo:
- Crear mascota
- Consultar por ID
- Actualizar a sold
- Consultar por estado

## 2. Tecnologías
- Java 17
- Maven 3.9.x
- Karate 1.4.1
- JUnit 5

## 3. Requisitos
- Java 17 instalado
- Maven instalado
- Conexión a internet

## 4. Verificar versiones
```bash
java -version
mvn -version
```

## 5. Ejecutar pruebas
```bash
mvn test
```

## 6. Ejecutar desde IDE
- Ejecutar PetStoreRunner.java
- o flujo-completo.feature

## 7. Reportes
